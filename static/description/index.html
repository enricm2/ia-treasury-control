IA Treasury Control — MCP

Manage your company's treasury, taxes, invoices and timesheets by talking to Claude.ai in plain language. Your Odoo data, accessible through AI.

What is it?

IA Treasury Control installs a lightweight MCP (Model Context Protocol) server inside your Odoo instance. Claude.ai connects to it and can query your financial data, create invoices, reconcile bank movements, log timesheets, and generate tax reports — all from a simple conversation.

Secure by design: The installed module contains no business logic. All agent processing runs on Uniasser's servers using your Anthropic API key — your key, your tokens, your control.

Key Features

💰 Treasury Reports

Pending receivables, payables, and cash flow forecast with AI-generated narrative.

🧾 Tax Status

VAT collected, VAT deductible, and IRPF withholdings — broken down by quarter.

📄 Invoice Creation

Create draft invoices from natural language. Always a draft — never confirmed without your approval.

🏦 Bank Reconciliation

Automatic matching proposals between bank movements and open invoices. You approve before anything is applied.

⏱ Timesheets

Log hours on projects and tasks, create projects and tasks — all from a chat message.

🔔 Financial Alerts

Overdue invoices, upcoming tax deadlines, and negative cash flow warnings consolidated in one call.

Technical Architecture

The module acts as a thin MCP/OAuth server inside Odoo. When Claude calls a tool:

  1. The Bearer token is validated locally against your configured MCP token
  2. Your license key, Odoo API key and Anthropic key are read from ir.config_parameter
  3. A secure HTTPS request is sent to apps.uniasser.net/agents/execute
  4. The remote server validates your license, connects to your Odoo via XML-RPC, executes the agent, and returns the result
  5. The result is returned to Claude.ai in MCP format
Privacy: Only the data required for each specific operation is accessed. The module uses read-only operations for reports and explicitly creates drafts (never confirmed records) for invoice creation.

Prerequisites

  • An active Odoo instance (16, 17, 18, 19 or Odoo.sh) accessible from the internet
  • A license key from apps.uniasser.net
  • An Anthropic API key from console.anthropic.com (Claude access)
  • Administrator access to your Odoo instance

Step 1 — Install the Module

Option A — Standard Odoo (ZIP upload)

  1. Download the ZIP file matching your Odoo version from apps.uniasser.net or apps.odoo.com
  2. In Odoo go to Settings → Apps
  3. Click Upload App and select the downloaded ZIP
  4. Search for "IA Treasury Control" and click Install

Option B — Odoo.sh

  1. Go to your Odoo.sh project → Branches → Production
  2. In the Code tab, click Add a module
  3. Enter the GitHub repository URL: https://github.com/enricm2/ia-treasury-control
  4. Select the branch matching your Odoo version (e.g., 16.0, 17.0)
  5. In Odoo go to Settings → Apps, search for "IA Treasury Control" and install

Option C — Command Line (self-hosted)

# Copy module to your addons path
cp -r ia_agents_treasury_control /opt/odoo/custom-addons/

# Update Odoo addons list
./odoo-bin -c /etc/odoo.conf -u ia_agents_treasury_control --stop-after-init

Step 2 — Configure the Module

Go to Settings → IA Treasury Control (scroll down in the Settings page).

2.1 — Enter your License Key

1 In the License Key field, enter your license key (format: XXXX-XXXX-XXXX-XXXX)
2 Click Validate License
3 The status should change to "Active"
Get your license at apps.uniasser.net. Free trial licenses are available.

2.2 — Generate an Odoo API Key

1 Go to Settings → Users & Companies → Users
2 Click on your user
3 Go to the API Keys tab → click New API Key
4 Enter a name (e.g., "MCP Treasury") and click Generate Key
5 Copy the key immediately — it will not be shown again
6 Paste it in Settings → IA Treasury Control → Odoo API Key
Important: Copy the API key immediately after generation. Once you close the dialog, it cannot be retrieved.

2.3 — Add your Anthropic API Key

1 Go to console.anthropic.com
2 Create an account and go to API Keys → Create Key
3 Copy the key (starts with sk-ant-)
4 Paste it in Settings → IA Treasury Control → Anthropic API Key

2.4 — Copy MCP Server URL and Token

In Settings → IA Treasury Control, copy the two values:

• MCP Server URL — looks like https://your-odoo.com/mcp
• MCP Token — a long random string used to authenticate Claude

Step 3 — Connect to Claude.ai

Option A — Claude.ai Web (claude.ai)

  1. Go to claude.ai and log in
  2. Click on your profile → Settings → Integrations
  3. Click Add Integration → Model Context Protocol (MCP)
  4. Fill in:
    • Name: Odoo Treasury (or any name you prefer)
    • MCP Server URL: paste the URL from Step 2.4
    • Authorization: Bearer <paste the MCP Token from Step 2.4>
  5. Click Save
  6. Start a new conversation and you'll see the Odoo tools available

Option B — Claude Desktop App (Mac / Windows)

Edit your Claude Desktop configuration file:

Mac: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json

{
  "mcpServers": {
    "odoo-treasury": {
      "url": "https://your-odoo.com/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_MCP_TOKEN_HERE"
      }
    }
  }
}

Replace https://your-odoo.com/mcp and YOUR_MCP_TOKEN_HERE with the values from Step 2.4. Restart Claude Desktop.

Auto-Install Scripts

Mac / Linux — Bash Script

Open Terminal and run:

#!/bin/bash
# IA Treasury Control — Auto-configure Claude Desktop
# Run: bash install_claude_mcp.sh

echo "=== IA Treasury Control — Claude Desktop Setup ==="
read -p "Enter your Odoo URL (e.g. https://myodoo.odoo.com): " ODOO_URL
read -p "Enter your MCP Token: " MCP_TOKEN

ODOO_URL="${ODOO_URL%/}"   # remove trailing slash
MCP_URL="${ODOO_URL}/mcp"

if [[ "$OSTYPE" == "darwin"* ]]; then
    CONFIG_DIR="$HOME/Library/Application Support/Claude"
else
    CONFIG_DIR="$HOME/.config/Claude"
fi

mkdir -p "$CONFIG_DIR"
CONFIG_FILE="$CONFIG_DIR/claude_desktop_config.json"

cat > "$CONFIG_FILE" <<EOF
{
  "mcpServers": {
    "odoo-treasury": {
      "url": "${MCP_URL}",
      "headers": {
        "Authorization": "Bearer ${MCP_TOKEN}"
      }
    }
  }
}
EOF

echo "✅ Config written to: $CONFIG_FILE"
echo "   MCP URL: $MCP_URL"
echo ""
echo "Restart Claude Desktop to apply changes."

Save this as install_claude_mcp.sh, then run: bash install_claude_mcp.sh

Windows — PowerShell Script

# IA Treasury Control — Auto-configure Claude Desktop
# Save as install_claude_mcp.ps1, then run in PowerShell

Write-Host "=== IA Treasury Control — Claude Desktop Setup ===" -ForegroundColor Cyan

$OdooUrl = Read-Host "Enter your Odoo URL (e.g. https://myodoo.odoo.com)"
$McpToken = Read-Host "Enter your MCP Token"

$OdooUrl = $OdooUrl.TrimEnd('/')
$McpUrl   = "$OdooUrl/mcp"

$ConfigDir  = "$env:APPDATA\Claude"
$ConfigFile = "$ConfigDir\claude_desktop_config.json"

if (-not (Test-Path $ConfigDir)) { New-Item -ItemType Directory -Path $ConfigDir | Out-Null }

$Config = @{
    mcpServers = @{
        "odoo-treasury" = @{
            url     = $McpUrl
            headers = @{ Authorization = "Bearer $McpToken" }
        }
    }
} | ConvertTo-Json -Depth 5

Set-Content -Path $ConfigFile -Value $Config -Encoding UTF8

Write-Host "✅ Config written to: $ConfigFile" -ForegroundColor Green
Write-Host "   MCP URL: $McpUrl"
Write-Host ""
Write-Host "Restart Claude Desktop to apply changes." -ForegroundColor Yellow

Troubleshooting

ErrorCauseSolution
"License not valid" Wrong key or expired Check the key at apps.uniasser.net. Make sure there are no extra spaces.
"Authentication failed" / 401 Wrong MCP Token Copy the token again from Settings → IA Treasury Control. No spaces.
"Cannot connect to Odoo" Odoo not reachable from internet Make sure your Odoo URL is accessible via HTTPS from outside your network.
"Odoo API Key invalid" Wrong or deleted API key Generate a new API key in Odoo → Users → API Keys and update the settings.
301 redirect error Using HTTP instead of HTTPS Make sure the MCP URL starts with https://
Tools not appearing in Claude MCP server not connected Check Settings → Integrations in claude.ai. Try removing and re-adding the integration.
"Module cannot be uninstalled" Module listed in server_wide_modules Remove ia_agents_treasury_control from server_wide_modules in your Odoo config file and restart Odoo.

Paso 1 — Instalar el módulo

Opción A — Odoo estándar (subida de ZIP)

  1. Descarga el ZIP para tu versión de Odoo desde apps.uniasser.net o apps.odoo.com
  2. En Odoo ve a Ajustes → Aplicaciones
  3. Haz clic en Subir aplicación y selecciona el ZIP descargado

Opción B — Odoo.sh

  1. Ve a tu proyecto Odoo.sh → Ramas → Producción
  2. En la pestaña Código, haz clic en Añadir módulo
  3. Introduce la URL del repositorio: https://github.com/enricm2/ia-treasury-control
  4. Selecciona la rama correspondiente a tu versión (p. ej. 16.0, 17.0)
  5. En Odoo ve a Ajustes → Aplicaciones, busca "IA Treasury Control" e instala

Opción C — Línea de comandos (autoalojado)

# Copiar módulo a la ruta de addons

Paso 2 — Configurar el módulo

Ve a Ajustes → IA Treasury Control (desplázate hacia abajo en la página de Ajustes).

2.1 — Introducir la clave de licencia

1 En el campo Clave de licencia, introduce tu licencia (formato: XXXX-XXXX-XXXX-XXXX)
2 Haz clic en Validar licencia
3 El estado debe cambiar a "Activa"
Obtén tu licencia en apps.uniasser.net. Hay licencias de prueba gratuitas disponibles.

2.2 — Generar una clave API de Odoo

1 Ve a Ajustes → Usuarios y Compañías → Usuarios
2 Haz clic en tu usuario
3 Ve a la pestaña Claves API → haz clic en Nueva clave API
4 Escribe un nombre (p.ej. "MCP Treasury") y haz clic en Generar clave
5 Copia la clave inmediatamente — no se mostrará de nuevo
6 Pégala en Ajustes → IA Treasury Control → Clave API de Odoo
Importante: Copia la clave API justo después de generarla. Una vez cierres el diálogo, no podrás recuperarla.

2.3 — Añadir la clave API de Anthropic

1 Ve a console.anthropic.com
2 Crea una cuenta y ve a API Keys → Create Key
3 Copia la clave (empieza por sk-ant-)
4 Pégala en Ajustes → IA Treasury Control → Clave API de Anthropic

2.4 — Copiar URL y Token del servidor MCP

En Ajustes → IA Treasury Control, copia los dos valores:

• URL del servidor MCP — tiene el formato https://tu-odoo.com/mcp
• Token MCP — una cadena larga aleatoria para autenticar a Claude

Paso 3 — Conectar con Claude.ai

Opción A — Claude.ai Web (claude.ai)

  1. Ve a claude.ai e inicia sesión

    • URL del servidor MCP: pega la URL del Paso 2.4
    • Autorización: Bearer <pega el Token MCP del Paso 2.4>
  5. Haz clic en Guardar
  6. Inicia una nueva conversación y verás las herramientas de Odoo disponibles
